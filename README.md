ecommerce-ui-regression-suite
Automated UI regression testing framework using Python, Selenium WebDriver and Pytest.

Features
Automated UI regression testing
Page Object Model (POM) design pattern
Pytest test execution
Reusable test setup with conftest.py
Technologies
Python
Selenium WebDriver
Pytest
Git
GitHub
Project Structure
pages/ # Page Object Model classes tests/ # Automated test cases conftest.py # Shared test configuration and fixtures requirements.txt

Installation
pip install -r requirements.txt

Run the Tests
pytest tests/

Purpose
This project was created to demonstrate practical UI test automation skills using Selenium and Pytest as part of a software QA portfolio.
